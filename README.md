### ✨ Features
- **Pair Code Login** - Input your number → Get code → Link device instantly
- **Bot Connected** - Runs 24/7 with zero downtime
- **Simple Setup** - No complex config needed
- **Lightweight** - Built with Node.js

### 🚀 Quick Start

1. **Get the code from bot**
   Click below and send `/start` to your number:
   [👉 Open @Gpraisebot](http://t.me/Gpraisebot)

2. **Link your device**  
   Input the pair code you got from the bot

3. **Done** ✓  
   Your device is now connected
